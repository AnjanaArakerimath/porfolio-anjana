# Portfolio - Dr. Anjana Arakerimath

Personal academic portfolio website featuring professional experience, academic activities, industrial visits, research publications, and personal galleries.

## 🚀 How to Deploy on Vercel

Since this is a static HTML/CSS/JS website, deploying to Vercel is incredibly fast and completely free. 

### Step 1: Connect your GitHub Account to Vercel
1. Go to [Vercel.com](https://vercel.com/) and sign up or log in using your GitHub account.
2. Once logged in, go to your dashboard and click the **Add New...** button, then select **Project**.

### Step 2: Import the Repository
1. In the "Import Git Repository" section, you will see a list of your GitHub repositories.
2. Locate `porfolio-anjana` and click the **Import** button next to it. (If you don't see it, you may need to click "Adjust GitHub App Permissions" and allow Vercel access to this repository).

### Step 3: Configure and Deploy
1. **Project Name:** You can leave the default name or change it to something like `anjana-portfolio`.
2. **Framework Preset:** Leave it as `Other` (Vercel will automatically detect that it's a static HTML site).
3. **Root Directory:** Leave it as `./`.
4. Click the **Deploy** button.

### Step 4: Your Site is Live!
Vercel will take a few seconds to build and deploy your site. Once it's done, you'll be provided with a live URL (e.g., `anjana-portfolio.vercel.app`) where your portfolio can be viewed by anyone globally!

*Note: Any time you push new code to the `main` branch on GitHub, Vercel will automatically update your live website instantly.*
